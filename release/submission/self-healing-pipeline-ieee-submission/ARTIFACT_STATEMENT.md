# Artifact Statement

This submission includes a reproducibility-oriented artifact package for the self-healing data pipeline research paper.

## Artifact Availability

The implementation artifact includes code, experiment scripts, generated figures, configuration files, and documentation needed to reproduce the reported evaluation workflow. The artifact is designed to support independent inspection of the failure injection, detection, classification, recovery, telemetry, and figure-generation process.

## Data Availability

The experiments use synthetic/generated pipeline data. No confidential enterprise data, customer records, insurance records, healthcare records, financial records, or personally identifiable information are included.

## Included Research Materials

- IEEE-format paper PDF
- LaTeX source files
- BibTeX bibliography
- Paper figures
- Tables and section files
- Reproducibility checklist
- Package manifest and checksums

## Reproducibility Scope

The package supports verification of the paper source, figure assets, bibliography, and artifact documentation. Full experiment reproduction should be performed from the project repository using the documented Python environment, tests, and experiment scripts.
