# Self-Healing Data Pipeline Research Paper — IEEE Submission Package

This folder contains the clean release/submission package for the paper:

A Self-Healing Data Pipeline Framework for Failure Detection, Diagnosis, and Automated Recovery

## Package Contents

- self-healing-pipeline-paper.pdf — compiled IEEE-format paper PDF
- main.tex — main LaTeX source file
- references.bib — BibTeX bibliography
- sections/ — LaTeX section files
- tables/ — LaTeX table files
- figures/ — paper figures
- ARTIFACT_STATEMENT.md — artifact availability statement
- REPRODUCIBILITY_CHECKLIST.md — reproducibility checklist
- MANIFEST.txt — file manifest
- CHECKSUMS-SHA256.txt — SHA256 checksums

## Build Notes

The paper was prepared using the IEEE LaTeX format. The clean package intentionally excludes temporary LaTeX build files such as .aux, .log, .out, .blg, and build folders.

## Reproducibility Summary

The research artifact uses synthetic/generated data, deterministic experiment scripts, and reproducible figure-generation workflows. No private, proprietary, or personally identifiable data is included in this package.
