Dear Editor / Program Chair,

Please consider our manuscript titled "A Self-Healing Data Pipeline Framework for Failure Detection, Diagnosis, and Automated Recovery" for review.

The paper presents a reproducible framework for improving data pipeline reliability through failure detection, diagnosis, automated recovery, and evaluation using synthetic pipeline experiments. The work addresses an important challenge in modern data engineering: maintaining reliable pipeline execution in the presence of source failures, schema anomalies, data quality degradation, and operational disruptions.

The submission includes a clean IEEE-format manuscript package, LaTeX source, BibTeX bibliography, figures, an artifact statement, and a reproducibility checklist. The evaluation uses synthetic/generated data and does not include confidential, proprietary, or personally identifiable information.

We believe the manuscript will be of interest to researchers and practitioners working in data engineering, data reliability, MLOps, pipeline observability, and automated recovery systems.
