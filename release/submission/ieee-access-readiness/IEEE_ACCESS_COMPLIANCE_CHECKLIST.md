# IEEE Access Compliance Checklist

## Template and Files

- [ ] Official IEEE Access template downloaded or verified
- [ ] Manuscript converted to IEEE Access format
- [ ] LaTeX source builds successfully
- [ ] Final PDF generated from matching source
- [ ] Figures render correctly
- [ ] Tables render correctly
- [ ] References render correctly
- [ ] No temporary build artifacts included in submission package

## Manuscript Content

- [ ] Title confirmed
- [ ] Abstract strengthened
- [ ] Keywords confirmed
- [ ] Introduction expanded
- [ ] Related Work expanded
- [ ] Methodology expanded
- [ ] Evaluation expanded
- [ ] Discussion added or strengthened
- [ ] Limitations expanded
- [ ] Artifact Availability strengthened
- [ ] Conclusion strengthened

## IEEE Access-Specific Checks

- [ ] Source file and PDF both prepared
- [ ] Supplemental material policy reviewed
- [ ] Code/artifact ZIP prepared if allowed
- [ ] AI-assistance disclosure reviewed
- [ ] Copyright/license requirements reviewed
- [ ] APC accepted or funding source identified
- [ ] IEEE Author Portal account ready
- [ ] Submission checklist reviewed from official site

## Research Integrity

- [ ] No private data included
- [ ] No personally identifiable information included
- [ ] Synthetic data statement included
- [ ] Data availability statement included
- [ ] Code/artifact statement included
- [ ] Conflict of interest statement ready
- [ ] Funding statement ready
- [ ] AI-assistance statement ready

## Final Before Submission

- [ ] Open final PDF manually
- [ ] Confirm author name
- [ ] Confirm author email
- [ ] Confirm affiliation
- [ ] Confirm ORCID if used
- [ ] Confirm all citations are referenced
- [ ] Confirm no broken figure references
- [ ] Confirm no LaTeX warnings requiring action
- [ ] Confirm final package ZIP opens correctly
