# Official Venue Verification Checklist

## Required Official Checks

For each venue, verify the following from official venue pages only:

- [ ] Aims and scope
- [ ] Article type
- [ ] Submission template
- [ ] Page limit or length expectation
- [ ] PDF/source file requirements
- [ ] Supplementary artifact policy
- [ ] Data/code availability requirements
- [ ] AI-assistance disclosure policy
- [ ] Conflict/funding/data statements
- [ ] APC or publication cost
- [ ] Copyright/license terms
- [ ] Peer review model
- [ ] Submission-to-decision timeline
- [ ] Indexing/visibility
- [ ] Preprint policy
- [ ] Desk-reject risk

## Paper-Specific Fit Checks

- [ ] Does the venue accept data engineering reliability work?
- [ ] Does the venue accept synthetic-data evaluation?
- [ ] Is the contribution framed as research, software, or applied engineering?
- [ ] Is the current 8-page IEEE version sufficient?
- [ ] Does the paper need expansion before submission?
- [ ] Does the paper need venue-specific template conversion?
- [ ] Does the artifact ZIP meet supplementary file requirements?
- [ ] Is the APC acceptable?
