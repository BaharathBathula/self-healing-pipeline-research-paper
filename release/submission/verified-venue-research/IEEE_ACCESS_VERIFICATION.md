## Verified Requirements and Observations

IEEE Access is a broad-scope, multidisciplinary, fully open-access IEEE journal. It emphasizes IEEE fields of interest, multidisciplinary topics, and applications-oriented engineering articles.

Submission requires the IEEE Access template. A Word or LaTeX source file and a matching PDF file are required at submission.

IEEE Access allows supplemental material such as code and data. It also references Code Ocean and reproducibility-related options.

AI-generated text usage must be disclosed according to IEEE Access submission guidelines.

Short author biographies are required for all authors in the article submission.

The listed APC is USD 2,160 plus applicable local taxes. The official page states there is no hard page limit, but recommends keeping articles under 20 pages for readability.
